# Sistema de Orientación Profesional TI

## Descripción

Sistema de orientación vocacional desarrollado en Python que analiza las habilidades, preferencias y competencias de estudiantes de informática para recomendar un perfil profesional dentro del área tecnológica.

El sistema utiliza diferentes paradigmas de programación: Programación Orientada a Objetos, Programación Funcional, Programación Lógica y Ciencia de Datos con Pandas.

---

## Características principales

- Test vocacional interactivo en consola con escala Likert (1 a 5).
- Motor lógico de inferencia basado en reglas para recomendar roles profesionales.
- Cálculo de afinidad mediante funciones de programación funcional.
- Identificación de habilidades críticas a fortalecer.
- Generación de reportes personalizados.
- Almacenamiento de resultados mediante archivos CSV.
- Análisis estadístico de evaluaciones usando Pandas.

---

## Roles profesionales evaluados

El sistema analiza perfiles como:

- Backend Developer
- Frontend Developer
- QA Automation Engineer
- Data Engineer / BI Analyst
- UX/UI Product Designer
- DevOps / Cloud Engineer
- Cybersecurity Blue Team Analyst
- IT Project Manager / Scrum Master

---

## Tecnologías utilizadas

- Python 3.x
- Pandas
- Archivos CSV
- Programación Orientada a Objetos
- Programación Funcional
- Motor de reglas lógico
